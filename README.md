TravelEase – Travel Booking Web Application

TravelEase is a single-page travel booking web application developed using HTML, CSS, JavaScript, and Bootstrap.
It allows users to register, log in, search travel packages, book tours, and view their bookings.
All functionality is handled on the client side using JavaScript and browser localStorage.

This project is suitable for college mini projects and frontend development practice.

Project Description

TravelEase simulates a real-world travel booking platform.
Users can create an account, log in, search for travel packages, select destinations, choose travel dates, make bookings, and simulate payments.

No backend or database is used. All data is stored temporarily in the browser.

User searches for travel packages

Payment is simulated

Booking is saved in browser localStorage

How to Run the Project

Download or clone the repository

Open the index.html file in any modern web browser

No server or installation is required
